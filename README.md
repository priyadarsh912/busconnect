Our project utilizes a modern, production-ready stack focused on speed and real-time synchronization:

Core Framework: React (v18.3) with TypeScript and Vite (SWC) for high-performance development.

UI & Styling: Tailwind CSS (v3.4), shadcn/ui, and Framer Motion for a premium, animated user experience.

State & Data: Supabase for the backend and real-time database, paired with TanStack React Query (v5) for asynchronous state management.

Maps & Visualization: Leaflet for interactive tracking and Recharts for data-driven authority insights.

Utilities: Zod and React Hook Form for data validation, and xlsx (SheetJS) for handling historical transit data and crowd predictions.

Features: react-qr-code for digital ticketing and Firebase Cloud Messaging for real-time push notifications.
